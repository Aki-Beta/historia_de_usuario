# Daily Sales Record System
## Description

This project is a simple sales record system written in Python.
It is a program that helps a store register products and calculate daily sales.

The system stores the product name, price, and quantity entered by the user.
It also generates a report showing the total quantity sold and the total revenue.

## How it works

The program creates an empty list called products.

The system asks the user to enter the product name, price, and quantity.

The function stores the product information in a dictionary.

The program adds the dictionary to the products list.

The system calculates the total sales using the price and quantity of each product.

The program prints a daily sales report with the total revenue.

## Results

The program is currently generating a daily sales report based on the products entered by the user.
